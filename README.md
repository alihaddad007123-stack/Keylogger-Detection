# Keylogger Detection Script 🔍

An educational tool for detecting suspicious keywords related to keyloggers in text-based files.

## How to Use

1. Run the script `KeyloggerScanner.py` in Python.
2. Enter the folder path you want to scan.
3. The script will scan **text files only** (.txt, .log, .ps1, .sh, .conf) for suspicious keywords.
4. Any detected keyword will be printed to the console and saved in `keylogger_report.txt`.
5. The report also shows the **most frequent keywords** and the **Top Suspicious File**.

## Notes
- This is an educational project for practicing file analysis, keyword detection, and malware scanning techniques.
